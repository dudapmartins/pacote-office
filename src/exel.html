Exel

O Microsoft Excel é um software de planilha eletrônica desenvolvido pela Microsoft. Ele é amplamente utilizado em ambientes corporativos, educacionais e pessoais para criar e manipular dados em formato de tabela. Com o Microsoft Excel, é possível criar planilhas de cálculo que permitem organizar, calcular e analisar dados de diferentes formas. Ele permite a criação de fórmulas e funções matemáticas, bem como a geração de gráficos e tabelas dinâmicas para visualização dos dados. O Excel também oferece recursos para importação e exportação de dados de diferentes formatos, como arquivos CSV e XML.

O Excel é amplamente utilizado em ambientes corporativos para controle de estoque, finanças, contabilidade e outras áreas que requerem manipulação e análise de dados em larga escala.

Comando Atalho no computador
Para inserir colunas e linhas. Ctrl + Shift + =
Para excluir colunas ou linhas. Ctrl + -
Para aplicar a formatação de hora. Ctrl + Shift + @
Para aplicar formatação de vírgula. Ctrl + Shift + !
Para aplicar a formatação de moeda Ctrl + Shift + $
Para aplicar formatação de porcentagem Ctrl + Shift + %
Para ir para a última célula em uma pasta de trabalho. Ctrl + End
Para adicionar ou editar um comentário de célula. Shift + F2
Para selecionar uma cor de preenchimento Alt + H + H
Para ocultar as linhas selecionadas Ctrl + 9
